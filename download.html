This program is provided only as source form.
It is written in C (not C++), using GTK 2.0 for the user-interface.
It should compile w/o problem on any Linux Distribution and FreeBSD, as long you have GTK 2.0 development installed.

To download the source, use svn:
git clone git@github.com:osingla/freelife.git

To compile the application, just follow these steps:
cd code/freelife/Release
make
